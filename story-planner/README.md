# Story Planner 📅

Aplikasi perencanaan konten YouTube & Shorts dengan Firebase.

## Fitur
- 📅 Kalender Konten
- 📝 Narasi & Scene
- 💡 Ideas Board
- 💬 Caption Draft
- 🔐 Prompt Vault
- 📔 Jurnal Analisis

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Konfigurasi Firebase
Copy `.env.example` ke `.env.local` dan isi dengan konfigurasi Firebase kamu.

### 3. Jalankan development server
```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000)

## Deploy ke Vercel

1. Push ke GitHub
2. Import project di Vercel
3. Tambahkan Environment Variables dari `.env.local`
4. Deploy!
